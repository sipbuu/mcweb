CBHS Minecraft Server

bedrock players guide

this page is just for minecraft: bedrock edition players (mobile, console, windows with the bedrock client). you’ll be connecting through geyser to reach the cbhs java server. keep in mind, you will mainly be able to play the vanilla servers, not the modded ones.

quick info

server address

Bedrock IP: bed.cbhs.online
Port: 19132

This works for mobile Bedrock and for some console setups that use Geyser.

sign in requirement

When you connect from Bedrock, you should see a prompt like: "Log in with a Microsoft account". You must log in with your normal Xbox / Microsoft account in order to join the CBHS server through Bedrock.

what’s happening?

We use Geyser so that Bedrock players can connect to a Java server. Your Bedrock account stays Bedrock; you’re just being “translated” into the Java server.

mobile (phone / tablet) setup

If you’re on iOS or Android using the normal Minecraft Bedrock app, follow these steps:

1

Open Minecraft Bedrock

Launch Minecraft and make sure you’re signed in with your Microsoft / Xbox account.

2

Go to "Play" → "Servers" or "Friends"

On some versions you’ll have an "Add Server" button. On others, Geyser / the Bedrock proxy may show up under "Friends".

3

Add the CBHS server

If you can add a server manually, use:

Server name: CBHS Minecraft
Server address: bed.cbhs.online
Port: 19132

4

Log in with Microsoft account

When Geyser connects you, you should see a "Log in with a Microsoft account" screen. Follow the on-screen steps to confirm your Microsoft account and finish linking.

After that, you should land in the CBHS lobby just like a Java player.

common issues & tips

Make sure you’re actually connecting through the Bedrock / Geyser address bed.cbhs.online and not the Java IP. Also confirm that you’re signed in to Minecraft with a Microsoft/Xbox account on that device.

CBHS server might be offline or restarting. Check the Announcements page or the Discord for status. On mobile, try fully closing Minecraft and reopening it before reconnecting.
