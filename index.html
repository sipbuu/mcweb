how to join?

follow these simple steps to join the server

1

Open Minecraft: Java Edition (or bedrock, we dont mind)

If you're using java, use version 1.21.x for the best experience. Other close versions may still work.

2

Add the Server

Go to Multiplayer → Add Server and use: mc.cbhs.online as the address or bedrock.cbhs.online for bedrock.

3

Read the Rules

Check the rules page so you know what’s allowed and how to not be a nuisance.

4

Join the Discord

Get updates, report issues, and hang out w/ other players in the CBHS Gaming Discord.
